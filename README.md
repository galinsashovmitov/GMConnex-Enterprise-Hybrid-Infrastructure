# GMConnex Enterprise Hybrid Infrastructure

## Project Overview

Enterprise hybrid infrastructure project built from scratch using VMware, Windows Server and Microsoft Azure.

The purpose of this project is to design, implement and document a production-inspired environment while following Microsoft and VMware best practices.

---

## Project Goals

- Build a production-inspired hybrid infrastructure
- Improve Windows Server knowledge
- Master VMware administration
- Learn enterprise networking
- Improve PowerShell automation
- Prepare for AZ-801, AZ-140 and CCNA
- Build a professional GitHub portfolio
